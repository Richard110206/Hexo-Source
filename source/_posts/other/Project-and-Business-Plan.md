自大学入学以来，通过参与各类竞赛以及与实验室不同专业不同年级的同学深入交流，我接触到了众多富有挑战性的项目，视野得以拓宽，思维也进一步解放。与此同时，层出不穷的新技术，仿佛一根根点燃的导火索，不断串联起我的思考，激发了一个又一个项目构想与创业计划。因此，我希望在此对这些 idea 进行较为系统的梳理与记录，内容包括：产生该创意的缘由、其价值所在、具体实现路径；或是分析可能面临的阻碍与技术壁垒，探讨为何当前难以落地；又或者，如果项目顺利推进，也将同步记录其最新进展！

### 校园知识库(`10.15`)

#### 一、项目缘由

在校园生活中，信息不对称与碎片化问题长期困扰师生：微墙作为校园高频交流渠道，历经多次迭代后，大量关于选课攻略、宿舍报修、考试安排等有价值的问答内容被新信息覆盖，隐性知识严重流失；师生获取信息需辗转于数十个群聊、教务官网、学院通知栏等多个渠道，信息噪音大、搜索效率低 。

偶然间，“速通 RAG” 的概念频繁出现在视野中（小学同学朋友圈分享 + 实习讲座接触），让我意识到这项技术恰好能解决校园信息整合难题。RAG（检索增强生成）技术可将碎片化信息转化为结构化知识库，结合 AI 问答功能实现精准响应，由此萌生构建校园即时问答小程序的想法。

#### 二、盈利方式

项目核心定位为校园公益服务，盈利模式以 “轻量、不影响用户体验” 为原则，优先保障知识库的纯净性与实用性：

1. **校园周边商家精准推广**：面向校园周边合规商家（如书店、文具店、培训机构、实习招聘企业等），提供精准广告投放服务（如在相关问答场景中嵌入合规推广信息），收取适度广告费。

2. **高校联盟授权推广**：待项目在本校成熟运营后，将产品模式复制推广至其他高校，通过 “授权使用 + 技术服务费” 的方式盈利，同时可组建高校知识库联盟，实现知识资源共享。

3. **技术成果转化与参赛获奖**：作为优质学生实践项目，参与各类创新创业大赛，争取获奖奖金；若项目表现突出，可吸引天使投资或创业孵化资源，推动项目进一步商业化落地。

#### 三、项目难点 / 技术壁垒

1. **付费意愿与技术效果双重制约，盈利落地难**：学生群体本身付费意愿普遍较低，而 RAG 技术实际应用效果存在不确定性，可能无法达到预期的精准问答效果，学生群体可能更加难以买单，盈利目标落地存在一定困难。

2. **提问推高 Token 消耗，成本控制承压**：RAG 模型运行过程中会产生 Token 消耗，尤其在用户高频提问、提问内容复杂的场景下，Token 消耗量可能持续增加，对成本控制构成挑战。

3. **校园信息收集与清洗难度大，需辅助辨别机制**：校园有效信息分散在微墙、各类群聊、论坛等多个渠道，其中微墙多为小程序形态，爬取难度较高；且整体数据量庞大，后续数据清洗工作复杂 —— 需筛选剔除恋爱贴、抱怨贴等无关信息，同时辨别并处理主观程度高、可信度低或已过时的内容。（可考虑采用 “时间戳 + 信息来源标注” 的方式呈现内容，供用户自行辨别参考，仅作为辅助信息提供）

#### 四、实施步骤

1. **多渠道数据采集阶段**：以校园微墙、官网文件、公开群聊为核心数据来源，定向采集与校园学习、生活、管理相关的问答、通知、经验分享等内容，构建初始数据池。

2. **数据清洗优化阶段**：运用大模型对采集的文本数据进行系统化清洗，重点过滤恋爱交友、无关吐槽、广告推广等无效信息，剔除重复内容与敏感信息，同时对剩余有效文本进行分词、关键词标注与语义梳理，提升数据质量。

3. **RAG 模型搭建与上线阶段**：基于清洗后的优质数据，在本地环境搭建 RAG（检索增强生成）系统，完成知识库构建与模型部署；随后开展多轮内部测试，验证问答精准度、响应速度与功能稳定性，根据测试结果优化模型参数与检索策略；待测试达标后，正式发布校园即时问答小程序，面向师生开放核心服务。

#### 五、项目进度

目前通过爬取 CUMT 生存手册网站的资料，已获得初步共计5w字的知识库，在cherry studio中进行测试，实测效果不错，在本地自己编写代码涵盖分块、嵌入等流程跑通 RAG 项目，未来计划尝试部署上线！



### 专利代理(`10.31`)

#### 一、项目缘由

实验室今年成功授权六项发明专利，最快授权周期仅 5 个月，远低于行业两三年的常规时长，且全国发明专利整体通过率仅 30%，我们的高效与高通过率，充分印证了团队在专利撰写、申请措辞优化及流程把控等方面的成熟经验。依托矿大专利申请官方账号资源，我们无需依赖外部代理机构 —— 相比市场上全包 12000 元、提供材料包协助申请 3000-4000 元的收费标准，具备显著成本优势与可盈利空间。

从市场需求来看，发明专利含金量突出：对学生而言，前三发明人可斩获智育素质分，亦是大学生创新项目结项的核心凭证；对教师群体，是职称评定、补贴申领的重要支撑，客源基础稳固。

#### 二、盈利方式

基于不同客户的需求差异，设计阶梯式服务体系：

1. **材料完善型服务（轻量指导类）**：针对已具备完整技术材料的客户，提供专利申请全流程精细化指导。核心服务包括专利文本语言措辞优化、权利要求书逻辑梳理、申请文件格式规范、官方系统申报操作指导及后续审查意见答复辅助，确保材料符合专利授权标准，提升通过率。
2. **从零定制型服务（全流程托管类）**：针对无现成技术材料或需从零开发专利的客户，由实验室机械部专业团队提供全流程定制服务。服务涵盖技术创新点挖掘、专利方案设计、图纸绘制、权利要求书及说明书撰写、申请流程全程代办，直至专利成功授权，为客户提供 “一站式” 专利解决方案，节省时间与精力成本。

#### 三、项目难点 / 技术壁垒

1. **资质合规性风险**：专利代理属于需行政许可的专业服务领域，需取得国家知识产权局认可的专利代理机构资质及代理人执业资格，而学生团队短期内难以通过专业考试获取相关资质。若以非正规代理身份开展业务，可能涉及 “非法代理专利” 相关法律风险，面临行政处罚。
2. **授权成功率的不确定性**：尽管实验室已有多项专利授权经验，但学生团队的专业知识广度、技术分析深度及对审查标准的把握精度有限，难以 100% 保证每一项专利都能通过审查并成功授权，若不处理好收尾工作容易引起分歧。

#### 四、项目进展
帮助大创中心的老师完成专利的撰写服务，若成功则是第一桶金，且老师会将我们的名声打出去，未来开阔市场！

### 专利代理全流程 AI WorkFlow(`11.7`)

#### 一、项目缘由

当时关注到 LLM 工作流（Workflow）相关实践，包括通过扣子（Coze）搭建自定义工作流、MCP 技术及多智能体的落地应用，深受启发。最初做专利代理工作时由昊天提出专利文档撰写过程中面临的实际痛点 —— 措辞规范难、图文对应费时费力等问题频发，而我结合自身专业特长与实验室积累的专利撰写经验，萌生了通过大模型微调（或 Prompt 工程优化）实现 AI 自主完成专利文档撰写的想法。

但思考过程又发现，传统专利代理仍需机械专业同学借助 SolidWorks 手动绘制结构图纸，流程存在断点，未能实现全链路自动化。于是想找找有没有能够实现AI绘图的方法，在 B 站找到 Cursor 插件与 SolidWorks 联动的相关视频，恰好找到了补全流程的关键突破口。

基于此，计划将 “AI 专利文档生成” 与 “AI 辅助机械结构绘图” 串联整合，构建一套完整的专利代理工作流：理想状态下，用户仅需提供专利标题（若能补充创新点等关键信息，输出会更精准详尽），系统即可自动完成机械结构透视绘图与专利权利要求书撰写。再叠加前面提到的专利代理服务，形成标准化、流水线式的专利产出模式 —— 不仅能大幅提升专利初稿的产出效率，还能让有经验的工程师聚焦于细节优化，进一步降低时间成本、提升整体工作效能！

#### 二、项目难点 / 技术壁垒

1. **全流程自动化搭建难度高，核心绘图环节受限明显**：当前 Cursor 插件与 SolidWorks 的联动效果难以达到预期，仅能完成抽屉、柜子等结构简单、形态标准化的机械模型绘制。对于包含创新点的复杂机械结构，现有工具组合无法精准识别设计逻辑，更难以生成符合专利绘图规范的透视模型，成为流程打通的关键卡点。

2. **“识图 - 撰写” 协同能力不足，跨模态匹配存在断层**：专利文档需精准关联机械结构的空间位置、连接关系等细节，这要求系统同时具备高效的图像识别能力去提取结构特征与撰写专业的文档。但目前 AI 的跨模态协同能力有限，易出现 “识图漏关键信息”“撰写与结构脱节” 等问题，导致输出文档的准确性、专业性难以达标。

3. **现有竞品先发优势下，差异化竞争力构建难度大**：通过市场调研发现，行业内已存在具备 “绘图 + 文档撰写” 相关功能的AI产品。如何将实验室团队长期积累的专利撰写经验转化为核心优势，形成与现有产品的差异化壁垒，避免陷入 “功能同质化” 竞争，是项目推向落地需突破的关键战略问题。

### 软著代理(`11.5`)

#### 一、项目缘由

核心逻辑与专利代理业务一致，依托本人已积累的软著申请实操经验，通过信息差转化形成服务价值。虽然远远不如专利的含金量高，但可作为大创结项的凭证，和参加创新创业比赛的辅助支撑材料，因此在目标群体中存在稳定的短期市场需求。当前行业定价，单个软著代理服务市场价为 1200 元，多个打包下单可优惠至 1000 元 / 个，存在较大盈利空间，可通过价格进行竞争。

#### 二、项目难点 / 技术壁垒

1. **主体适配风险**：当前仅完成个人名义的软著申请尝试，而目标客户的核心需求是将专利权人登记为学校，该场景下的申请流程尚未实际验证，存在流程适配、材料要求等未知风险。

2. **授权通过率不确定**：缺乏大量成功的案例作为参考，且软著审查标准存在一定弹性，无法精准评估服务的授权通过率，存在因材料不合规、技术描述不达标等导致驳回的潜在风险。

3. **加急通道缺失**：尚未打通官方加急申请渠道，相较于专业代理机构的快速审批，下证周期更长且存在不确定性，影响服务竞争力。


### README全流程AI(`11.16`)

#### 一、项目缘由

在 GitHub 探索开源项目时，常遇到这样的困境：明明被有趣的功能吸引，想要本地部署体验，却屡屡被 README 文件 “劝退”—— 要么是环境配置步骤模糊，依赖包版本冲突、系统兼容问题无迹可寻；要么是关键操作缺失，比如需要手动添加的配置文件、需调整的路径参数未被提及，导致开发者反复踩坑、耗时探索，最终仍可能无法成功运行项目。

而作为开源项目作者，我们同样面临两难：耗费大量精力开发的成果，却因 README 不够直观、不够详尽，或缺乏小白友好的指引，导致用户体验不佳，难以实现项目的广泛传播与落地；同时，手动打磨一份高质量 README 需投入大量时间，既要兼顾美观性，又要覆盖不同用户的操作场景，效率低下。

如今，AI 工具（如豆包）已具备强大的信息提炼与场景化指导能力 —— 输入项目链接或文件夹结构，即可自动解析核心逻辑、生成运行指南。因此希望借助 AI 的高效赋能，结合开发者的实战经验与踩坑总结，打造一套 “AI 初生成 + 人工精优化” 的 README 全流程解决方案，既大幅提升 README 制作效率，又确保其具备易懂性、完整性、可操作性，让小白无痛部署项目，让开源作者的成果被更多人认可。

### 全套创赛服务（`12.25`）

这个项目可以说是我感觉最有希望，资料内容最完善、可行性最强、综合性和可落地性最高的一个项目了！

####   一、项目背景
其实柏哥很早就有过这个项目的初步构想（或许说这本身也是创立实验室的目的），但一直苦于找不到合适的人选去实现这个大业而搁置，最近正所谓“说者无心，听者有意”，从旁人那里获得了一点灵感，仿佛打通了任督二脉，项目的产业链逻辑得以贯通，整体框架趋于完善！

当前大一、大二学生群体中，存在一组鲜明的核心矛盾：一方面是对通过竞赛加分实现保研、考研等升学目标的强烈需求；另一方面是获取优质竞赛项目资源、参与专业竞赛的渠道严重不足。我们正是窥见了这其中的二者相矛盾的商机，决心利用我们的经验与资源，为同学们提供创赛服务！

#### 二、服务覆盖领域

依托实验室核心资源，我们可提供全方位创赛服务指导，具体涵盖以下两大核心板块：
- **竞赛项目挂售服务**：按项目完整度分级提供，包括全要素完整项目（含专利、实物演示模型、商业计划书、PPT、软著）、半成品项目（具备上述部分要素）及创新想法项目（仅500字左右内容简介），可根据用户个性化需求匹配对应服务。
- **创赛指导服务**：涵盖各专业科研项目流程解析，支持项目主负责人自主招募队友，我们提供线上远程指导、项目进度管控等核心服务。以实验室项目运作标准为例，完整项目团队需配置机械组、电控组、软件组及商业组。具体流程如下：机械组基于创新想法检索知网相关专利，开展创新性设计与SolidWorks建模绘图，完成发明专利撰写以保障知识产权，再通过3D打印制作模型并完成组装；电控组同步配置电机等模块，赋予机械结构功能性，使其原本机械结构具有活的灵魂（不好意思，才考完的毛中特串台了）。与此同时，软件组的同学可以搭建展示平台（事实上，只需要有前端展示页面即可，甚至而言用PPT进行绘制效果也是类似的），如果是网站可能效力还不是那么够，因而可以尝试使用app（主要是安卓平台），小程序的方式（如果难度较大，甚至可以使用某些一站式部署的开源项目）；商业组结合项目核心内容与市场调研，撰写商业计划书、制作PPT，并制定针对性商业运营模式。至此通过全流程协同整个项目完全打通，可满足多数综合性创新创业赛事的参赛要求。

#### 三、核心竞争优势

当前市场中“挑战杯”、“互联网+”、数学建模包获奖、明码标价的不在少数，我们如何进行差异化竞争脱颖而出才是关键！依我看来主要分为：价格、服务两个维度！

- 价格方面：依托实验室现有的设备可以承接外来3D打印的项目，相比于外面的专业团队，我们更加倾向于提供项目服务，而并非完善的项目本身，因而价格上会跟更加的友善；与此同时，我们也拥有申请软著、专利的相关经验，可提供官费减免等附加服务，进一步强化价格竞争力，卷价格！
- 服务方面：实验室成员均具备完整项目运作及创新创业赛事参赛经验，可提供全流程一站式指导（在项目指导过程中同步嵌入软著、专利申报服务，同步解决专利业务的客户引流问题）；2025年度实验室成功授权6项国家发明专利，充分证明我们在专利撰写、申请、评估领域的专业能力；经过多年积淀，实验室储备了充足的优质项目资源，依托江苏211高校的生源质量优势，项目品质具备可靠保障，不仅可服务本校学生，更能向下覆盖双非院校（此类院校保研竞争激烈，竞赛加分需求更为迫切，市场前景广阔），实现优质项目资源的持续输出。

#### 四、实施步骤
